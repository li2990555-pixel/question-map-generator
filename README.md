# 问题导图生成器

边读社科人文书，边生成问题链，输出 **Obsidian Canvas 可视导图** + **HTML 静态图** 两种格式。

## 是什么

任何一本社科人文书，本质都是在回答一个或多个核心问题。这个 Skill 帮你边读边问，把知识点转化成「问题→回答→局限→下一问题」的因果链条，并生成可视化的问题导图。

## 适用类型

| 类型 | 核心问题模板 |
|------|------------|
| 文学史 | 历史条件 → 时代问题 → 思潮回答 → 局限 |
| 文学理论 | 既有理论困境 → 核心争辩 → 理论方案 → 盲点 |
| 哲学 | 既有哲学困境 → 前人哪错了 → 新框架 → 边界 |
| 历史 | 时代背景 → 核心矛盾 → 历史进程 → 启示 |
| 社会学 | 社会现象 → 背后结构 → 理论解释 → 解释边界 |

## 使用方法

在 WorkBuddy 中调用：

```
@skill:"问题导图生成器" @"/path/to/book.pdf" 我想开始阅读这本书
```

## 输出结构

每次更新都会同时生成两种格式的文件：

```
docs/问题导图/
  ├── {书名}/
  │   ├── {书名}.canvas          ← Obsidian Canvas 文件
  │   └── cards/
  │       ├── Q1.md              ← 问题卡片
  │       ├── A1.md              ← 回答卡片
  │       ├── L1.md              ← 局限卡片
  │       └── ...
  └── {书名}_问题导图.html       ← 浏览器直接打开的静态图
```

### Canvas 文件（Obsidian）

- 横向流 + 纵向堆叠，每轮 Q→A→L 从左到右展开
- 用 Group 节点包裹每轮，颜色交替区分
- 卡片用独立 `.md` 文件引用，Obsidian 完整渲染，不截断文字
- 需 Obsidian 打开使用

### HTML 静态图（浏览器）

- 白底卡片 + 彩色左边框（Q 棕红 / A 绿 / L 橙棕）
- 每轮横向 Q→A→L，轮间细线 +「引出」标签连接
- 含 `ROUND X · 章节` 标题，完整文字可直接阅读
- **无需任何软件，双击即用浏览器打开**
- 样式内联，离线也能正常显示

两种格式的文字内容均来自同一套 `cards/` 目录下的 `.md` 文件，修改卡片后重新生成即可同步。

## 安装

将 `SKILL.md` 放入 WorkBuddy 的 `.workbuddy/skills/` 目录即可。
